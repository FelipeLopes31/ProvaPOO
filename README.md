# ProvaPOO
#Prova de programação orientada objeto ( TYPESCRIPT)


Exemplo de POO utilizando Typescript.


## Documentação
[NodeJS](https://nodejs.org/en/)
[Typescript](https://www.npmjs.com/package/typescript)

## Instalação

Instale poo com npm

```bash
  npm install
  cd poo
  ts-node /path.arquivo.ts
```
    
## Usado por
